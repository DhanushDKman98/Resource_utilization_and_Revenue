This repository contains interactive Tableau dashboards built to analyze revenue performance and resource utilization. 
The goal of these dashboards is to turn raw data into clear, decision-ready insights using clean visuals, logical structure, and practical KPIs.

All dashboards are built using Tableau Desktop (.twb files) and are intended for exploratory analysis as well as executive-level reporting.

Revenue_perf_analysis.twb
This dashboard focuses on understanding how revenue behaves across time, categories, and key business dimensions.
What it covers:
Overall revenue trends over time
Performance breakdown by category / segment
Identification of high- and low-performing areas
Comparison of revenue contribution across dimensions

File: Resource_util_dash.twb
This dashboard analyzes how efficiently resources are being allocated and utilized.
What it covers:
Resource usage patterns
Utilization rates across teams / categories
Identification of underutilized or overutilized resources
Operational efficiency insights
